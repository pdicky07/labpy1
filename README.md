```python
a=int(input("Masukan bilangan a:"))

b=int(input("Masukan bilangan b:"))

c=int(input("Masukan bilangan c:"))

if a>b:

    maks=a
    
else:

    maks=b
    
if maks<c:

    maks=c

print("nilai terbesar=",maks)
```
